About FreeMind-MMX Web

This is a web port of the FreeMind-MMX concept. It separates the mind map data into two files:

  • .mm: Contains the structure and text (clean content).
  • .mmx: Contains volatile attributes like folded state (auxiliary data).

When saving, both files are generated. When opening, select both files to restore the full state.
